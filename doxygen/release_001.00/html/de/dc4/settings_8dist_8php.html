$tolc_conf['pref_use_prepared_statements'] = false

Using of prepared statements can prevent SQL injection attacks (even if user input is thoroughly sanitized). But, MySQL php extension dows not support prepared statements (only PDO and mysqli). So, this settings is for testing purposes and for future use of PostgreSQL or other RDBMS
